Compilation:
  On Linux run `make` in this directory.
  We did not try it on Windows and we do not care. Find out yourself.

--------------------------------------------------------------------------------

How to play:
  `make run` to run both, CLI and GUI version.

  Command line version:
    Run `./hra2016-cli` and all you will see all you need to know.

  Gui:
    Run `./hra2016`. The only game controller is mouse.

  Rules: https://en.wikipedia.org/wiki/Reversi

  Program offers two AI algorithms or you can play against real friend.

  Have a nice time playing.
